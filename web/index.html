1. How to Configure Database & Back-end on Your Server?

  1. First of all find the PHP / Quotes folder from the downloaded package and upload to your server.
  2. Create a new database from cPanel
  3. Create new user for the database & give all the privileges to the user
  4. Import the database file 'quotes.sql' to your PHPMYADMIN
  5. Open File Manager or FTP or File Explorer
  6. Go To 'library/' directory and Open 'crud.php' file in PHP Code
  7. set 'DOMAIN_URL' to your www.mydomain.com domain on line number - 8
    define('DOMAIN_URL','www.mydomain.com');
  8. Set database credentials with your credentials on line number - 17,18,19,20
    private $db_host = "localhost"; // Change as required
    private $db_user = "mydb_user"; // Change as required
    private $db_pass = "mydb_user_password"; // Change as required
    private $db_name = "myquotes_db"; // Change as required

Thats it! Now you are ready to run : http://www.mydomain.com
